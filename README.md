# Alizé — maquette

Prototype cliquable d'une application de **mise en relation de services à la demande**
à **Saint-Barthélemy** : ménage, garde d'enfants, jardinage et autres petits services
pour des besoins ponctuels.

Principe (façon Uber, à double sens) :

- le **client** enregistre une carte une fois et commande un service ;
- une **entreprise / un prestataire** vérifié reçoit la demande et l'accepte ;
- le montant est **bloqué en garantie (séquestre)** pendant la mission ;
- le prestataire n'est **payé qu'après validation** de la prestation par le client ;
- la plateforme prélève une **commission de mise en relation**.

## Utilisation

Ouvrir `index.html` dans un navigateur (aucun backend, aucune dépendance).
Basculer entre les onglets **Client** et **Entreprise** pour dérouler le scénario
de bout en bout. Données simulées, aucun paiement réel. « Alizé » est un nom de travail.

> Application indépendante — distincte du projet Archipel.
